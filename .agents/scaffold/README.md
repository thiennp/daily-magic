# Scaffolding (`pnpm scaffold`)

Use **`pnpm scaffold`** from the repository root to generate boilerplate for components, hooks, stores, enums, constants, layouts, types, mocks, utilities, tests, and styles.

Implementation lives in **`.agents/scripts/scaffold.ts`** (templates are embedded there). Prefer this over hand-written empty files; always-on conventions are in **`.cursor/rules/`** and **`CLAUDE.md`**.
