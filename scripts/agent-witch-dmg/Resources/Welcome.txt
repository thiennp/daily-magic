Agent Witch for Mac
===================

1. Double-click "Agent Witch Installer"
2. Click Install (first time) or Update (when already installed)
3. Open https://www.agentwitch.com and link this Mac to your account

You do not need to paste curl commands into Terminal.

Update uses ~/.agent-witch/self-update.sh when present (same as the hourly updater LaunchAgent).

Unsigned builds: if macOS blocks the app, right-click → Open, or run:
  xattr -cr "/Volumes/Agent Witch/Agent Witch Installer.app"
